switches

quick switches for jekyll
Published in 29 Jan 2020

With mobile browsers as our canvas, performance is always an issue. It’s crucial for the page to only load what it needs, and nothing more.

Jekyll uses frontmatter for page requirements. We created a page.switches variable, where each tag activates an if function on default.html layout, when listed.
